## 🚀 Proyectos Destacados

### 🎯 Capsule Care – Plataforma inteligente para el cuidado de medicamentos

[![Live Demo](https://img.shields.io/badge/-DEMO-000?style=for-the-badge&logo=vercel&logoColor=white)](https://capsule-care-demo.vercel.app)
[![Código Fuente](https://img.shields.io/badge/-Repositorio-181717?style=for-the-badge&logo=github)](https://github.com/rusbelteclas/capsule-care)

App móvil enfocada en el seguimiento de tratamientos médicos con:
- 🔔 Notificaciones automáticas
- 🧠 Integración con estándares médicos (FHIR)
- 📈 Panel de control intuitivo
- 🛡 Autenticación segura con Firebase

**Stack:** React Native · Firebase · Node.js · MongoDB
